시큐어코딩? 알고리즘?

시큐어코딩이란 안전한 소프트웨어 개발을
위해 소스 코드에 존재할 수 있는 잠재적인
보안 취약점을 제거하고 기능 설계 및
구현하는 활동을 일컫는 말입니다.

본 교육과정에서는 http와 웹 구조,
세션과 쿠키, 로그 관리와 로그분석,
입력값 검증 부재와 삽입,
세션과 인증 관리 등을 중심으로
시큐어코딩을 배웁니다.

알고리즘은 문제를 해결하는 단계를
의미합니다. 예를 들어 구글 지도에서
길찾기를 하면 가장 빠른 경로를 자동으로
알려주는데, 이때 적용된 것이
Route Finding Algorithm입니다.

많은 기업들이 알고리즘 코딩테스트를
통해 개발자를 채용하고 있으며,
그 수는 더 늘어날 것으로 전망됩니다.

자바 웹 기반 시큐어 코딩 전문과정 커리큘럼

본 수업은 여러분을 실력있는 웹 개발자로 만드는 것을 목표로 합니다. 여러분은 6개월 동안 정규 커리큘럼과 플레이데이터만의 특별한 특강을 통해 웹 개발의 A부터 Z까지 배우게 됩니다.

사전 온/오프라인 학습시스템
비전공자도 수업을 잘 따라갈 수 있도록 사전 교육을 진행합니다. 사전 교육은 희망자를 대상으로 무료로 진행되며, 코딩을 한번도 해보지 않은 분들은 수강을 권장 드립니다. 사전 교육을 통해 기초 프로그래밍을 배워보세요!

- [온라인 사전교육 (20일)]

- HTML5, CSS3 기초

- JavaScript 기본 문법

- MySQL : 테이블 생성, CRUD

- 알고리즘 : 순환, 정렬, 검색트리

- [오프라인 사전교육 (10일)]

- Python 개발 환경

- Python 기본 문법

- Python 데이터 타입, 기본 연산자

- Python 함수
Step 1 HTML5 웹 표준
Step1에서는 웹 사이트의 뼈대, 살, 근육이라 할 수 있는 HTML, CSS, Javascript를 배웁니다. 웹 표준을 지키는 문서 구조화, 반응형 프로그래밍, 동적 웹 사이트 제작을 위한 JavaScript와 백엔드 개발을 위한 Node.js를 배우실 수 있습니다.

- HTML5 기본 개념

- HTML5 태그

- CSS의 선택자와 레이아웃

- JavaScript와 HTML5 API

- Ajax : XMLHttpRequest 객체

- Ajax : 서버 응답

- Node.js : http 모듈과 웹서버
Step 2 자바 기초 프로그래밍
Step2에서는 Java를 배웁니다. Java는 현재 전 세계에서 가장 많이 사용되는, 가장 중요한 언어입니다. Java로 프로그래밍 기초실력을 탄탄하게 다지세요.

- 자바 개발환경 구축

- 기본 문법

- 객체 지향 프로그램

- 데이터베이스 연동

- 데이터베이스 연동

- GUI 프로그램

- 파일 입출력과 스레드

- 네트워크 프로그램
Step 3 자바로 배우는 알고리즘
알고리즘은 컴퓨터가 문제를 해결할 수 있도록 단계를 설계하는 것을 의미합니다. 알고리즘은 프로그래밍의 기본이며 이미 많은 기업들이 알고리즘 코딩테스트를 통해 개발자를 채용하고 있습니다.

- 검색 알고리즘

- 정렬 알고리즘

- 알고리즘 분석

- 리스트
Step 4 데이터베이스와 SQL
기업들이 웹 사이트를 만드는 이유는 고객과 소통을 하기 위해서고, 그 소통은 ‘데이터’라는 형식으로 흔적을 남깁니다. 그렇게 쌓이는 데이터의 양은 상상하기 어려울 정도로 크기 때문에 데이터베이스를 이용해 저장해야 합니다. SQL은 데이터를 저장하고 처리하고 관리할 수 있는 핵심 언어입니다.

- 관계형데이터베이스와 SQL

- SQL조회와 조인/서브쿼리

- SQL 데이터 조작어, 정의어

- PL/SQL

>
Step 5 웹 어플리케이션
JSP는 동적 페이지를 작성할 때 사용하는 자바의 표준 기술로서 HTML 응답을 생성하는데 필요한 기능을 제공합니다. JSP/Servlet을 배우면 웹 개발 기술의 전반을 이해하는 데 큰 도움이 됩니다.

- Servlet 프로그래밍

- JSP 프로그래밍
Step 6 스프링 부트
스프링 부트 툴을 사용하면 스프링 프레임워크를 더 쉽고 빠르게 사용할 수 있습니다. Step 6 에서는 스프링 부트의 원리와 사용법에 대해 배웁니다.

- 스프링 프레임 워트

- 스프링 부트 웹 개발

- REST API 서버 만들기

- 스프링 부트와 데이터

- 커스텀 스프링 부트

- 예외처리 및 테스트

- 배포
Step 7 시큐어 코딩
시큐어코딩이란 안전한 소프트웨어 개발을 위해 소스 코드 등에 존재할 수 있는 잠재적인 보안 취약점을 제거하고, 보안을 고려하여 기능을 설계 및 구현하는 활동을 일컫는 말입니다. Step 7 에서는 Http와 웹구조, 세션과 쿠키, 로그 관리와 로그분석, 입력값 검증 부재와 삽입, 세션과 인증 관리 등을 중심으로 시큐어코딩을 배웁니다.

- 안전한 소프트웨어 개발 방법론

- 시큐어 코딩 기법

- 보안 취약점 제거를 위한 코딩
