빅데이터 플랫폼이란?

일반적으로 데이터가 DB에 저장되어 있는 경우에는 SQL 쿼리를 작성한 후 엑셀로 추출하지만
데이터의 양이 상상하기 어려울 정도로 큰 경우에는 위와 같은 방식을 사용하면 오랜 시간이 소요됩니다.
빅데이터 플랫폼이란 이처럼 거대한 양의 데이터를 추출하기 위해 구축하는 인프라입니다.
Hadoop과 Spark는 빅데이터 플랫폼 구축에 사용되는 오픈소스 빅데이터 프레임워크로,
여러 개의 컴퓨터를 하나로 묶어 대용량 데이터를 분산 처리하는 역할을 합니다.

시큐어코딩? 알고리즘?

시큐어코딩이란 안전한 소프트웨어 개발을 위해 소스 코드 등에 존재할 수 있는
잠재적인 보안 취약점을 제거하고 기능을 설계 및 구현하는 활동을 일컫는 말입니다.
본 교육과정에서는 http와 웹 구조, 세션과 쿠키, 로그 관리와 로그분석,
입력값 검증 부재와 삽입, 세션과 인증 관리 등을 중심으로 시큐어코딩을 배웁니다.

알고리즘은 컴퓨터가 문제를 해결하는 단계를 의미합니다. 예를 들어 구글 지도에서 길찾기를 하면
가장 빠른 경로를 알려주는데, 이때 적용된 것이 Route Finding Algorithm입니다.
많은 기업들이 알고리즘 코딩테스트를 통해 개발자를 채용하고 있으며,

플레이데이터만의 특별한 실무특강!

1. 프로젝트 기획 방법론

모든 소프트웨어 개발의 시작은 ‘기획’입니다.
서비스 기획 실무자로부터 기획에 대해 배웁니다.

2. 소스코드 형상관리 Git 특강

개발은 주로 팀 단위로 진행합니다.
팀 프로젝트에서 코드 버전 관리는 필수입니다.
Git 소스코드 형상관리 방법을 학습합니다.

3. 프론트엔드 개발을 위한 Bootstrap 특강

Backend 개발자도 기본 Frontend 스킬은 필수!
CSS Framework인 Bootstrap으로
웹 화면을 구현하는 방법을 학습합니다.

4. 데이터베이스 모델링 특강

데이터 전문기업 엔코아의 현역 컨설턴트가
직접 오프라인 특강 후 온라인 멘토링을 진행합니다.

1. 인재 추천

우수 교육생에게 엔코아 협력사 및 연계 기업
채용 기회를 우선적으로 드립니다.

2. 기업 채용 설명회

기업 인사담당자에게 기업의 비전, 인재상,
분위기 등의 정보를 얻을 수 있습니다.

3. 데모데이

현직 개발자와 인사담당자가 데모데이의 심사위원으로
참석해 여러분의 프로젝트를 평가하고 시상합니다.

4. 이력서 코칭

희망하는 직무의 현업 개발자가 직접 자소서를
첨삭하고 취업 관련 질문에 답변합니다.

4. 이력서 코칭

희망하는 직무의 현업 개발자가 직접 자소서를 첨삭하고 취업 관련 질문에 답변합니다.
