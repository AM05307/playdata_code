기초를 다진 후에는 Hadoop, Spark 등을 이용해 빅데이터 플랫폼을 설계 및 구축하고,
빅데이터를 분산처리하는 법에 대해 배우게 됩니다.
또한 팀별 프로젝트를 진행하며 빅데이터 분석과 관련된 모든 과정을 실제로 경험합니다.

빅데이터 엔지니어란?

데이터 사이언스 분야는 크게 빅데이터 엔지니어와 빅데이터 분석가라는 직군으로 나누어집니다.
분석가는 잘 가공된 데이터를 통계적 기법 등을 활용해 분석합니다.
빅데이터 엔지니어는 분석가가 분석할 데이터를 만들어내는 일을 합니다.
대부분의 경우 데이터는 데이터베이스에 깔끔하게 정리되어 있지 않으며 파일의 형태로 로그에 저장되어 있거나
때로는 실시간으로 데이터를 처리해야 할 때도 있습니다.
빅데이터 엔지니어는 데이터를 효과적으로 가공하고 처리하기 위한 인프라를 설계/구축하고,
실제로 데이터를 수집/저장/처리/시각화하는 일을 합니다.

이 수업을 들으면 어떤 분야로 진출할 수 있나요?

빅데이터 엔지니어

빅데이터 엔지니어는 대용량 데이터 수집/관리/분석을 위한 분산시스템(플랫폼) 설계 및 개발,
데이터 기반 솔루션 개발 및 운영, 통합데이터 플랫폼 백엔드/프론트엔드 개발,
대용량 트래픽 처리를 위한 시스템 튜닝 및 오픈소스 최적화와 같은 업무를 담당합니다.

※ 채용공고 분석
자격요건 : JAVA / Python / Scala / Hadoop / Spark / SQL / Linux
우대사항 : ElasticSearch, Mongodb, AWS 사용 경험

플레이데이터 수료생 취업처

플레이데이터 수료생들이 일하고 있는 회사들입니다.
저희는 수강생 분들이 원하는 기업에 취직할 수 있도록 최선을 다해 지원하고 있습니다.

하지만 이것이 전부가 아닙니다. 저희는 플레이데이터가 교육기관인 동시에,
개발자들의 playground가 되기를 바랍니다.
이를 위해 선배와 후배가 만나 개발공부, 취업 등 다양한 정보를 공유할 수 있는
홈커밍데이, 선배와의 대화 같은 네트워킹 행사도 준비하고 있습니다.

플레이데이터를 거쳐가는 모든 수강생 분들이
장기적으로 성공적인 커리어를 쌓아갈 수 있도록 최선을 다하겠습니다.

저도 수업을 들을 수 있을까요?

네, 여러분도 할 수 있습니다.
십여 년 경력의 선생님이 프로그래밍의 기초와 빅데이터 플랫폼 설계 및 구축에 대해 체계적으로 알려드립니다.

일반 프로그래머에 비해 취업의 선택지가 넓습니다.

[자바 시큐어코딩 웹 개발자 양성과정] 또는 [클라우드 기술 기반 웹 개발자 양성과정]을 추천드립니다.
두 과정은 기초를 좀 더 깊이 가르치고, IT 기업에 취업하기 위해 꼭 필요한 알고리즘을 가르치기 때문입니다.

이 수업을 듣고 개발자로 취업하고 싶습니다. 가능할까요?

본 과정은 취업교육과정이며 플레이데이터도 이를 위해 다방면으로 노력하고 있습니다.

외부 인사담당자가 참여하는 데모데이 프로젝트 발표회와 여러 기업이 참여하는 채용설명회도 개최합니다.
온오프라인으로 개발자 채용정보를 제공하고 있으며,
이력서/면접 컨설팅과 자소서 첨삭 컨설팅 같은 취업지원 프로그램도 운영하고 있습니다.
