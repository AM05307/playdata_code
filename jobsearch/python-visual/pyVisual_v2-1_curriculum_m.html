사전 온/오프라인 학습시스템

수강생분들이 수업을 잘 따라갈 수 있도록 사전교육을 진행합니다. 사전교육은 희망자를 대상으로 무료로 진행되며 코딩을 한번도 해보지 않은 분들은 사전교육을 수강하시길 추천드립니다. 본 교육 전에 미리 프로그래밍을 배울 수 있습니다.

- [온라인 사전교육 (20일)]

- HTML5, CSS3, JavaScript

- MySQL, 알고리즘


- [오프라인 사전교육 (10일)]

- Python : 기본 문법, 데이터 타입, 연산자, 함수

Step 1 - 자바

프로그래밍의 세계에 오신 것을 환영합니다! Step1에서는 Java를 배웁니다. Java는 현재 전 세계에서 가장 많이 사용되는, 가장 중요한 언어입니다. Java로 프로그래밍 기초실력을 탄탄하게 다지세요.

- 자바 개발환경

- 기본 문법

- 클래스와 OOP

- 스레드

- IO

- 네트워크

Step 2 - 데이터베이스

SQL은 데이터를 저장하고 처리하고 관리할 수 있는 핵심 언어입니다.

- 오라클 개발환경

- DDL, DML

- 조인

- 트랜잭션

- 제약조건

- JDBC

Step 3 프론트엔드

Step3 에서는 웹 사이트의 뼈대, 살, 근육이라 할 수 있는 HTML, CSS, Javascript를 배웁니다.

- HTML

- CSS

- JavaScript

Step 4 스프링

Step4 에서는 프로젝트 단위의 웹 어플리케이션 개발을 위해 스프링 프레임워크에 대해 배웁니다.

- Jquery

- 스프링 환경설정

- DI, AOP, DAO

- MVC

- 활용 : Sprint 기반 게시판

- MyBatis

중간 프로젝트

지금까지 배운 모든 내용을 활용해서 중간 프로젝트를 진행합니다. 스프링 부트 기반의 웹 프로젝트를 만들어본 경험은 취업할 때 큰 도움이 되실 겁니다.

- UML, ERD

- 윈도우 : 스프링 프레임워크 기반의 웹프로젝트

Step 5 리눅스

리눅스는 무료 오픈소스 운영체제입니다. 개인용 컴퓨터에서 슈퍼 컴퓨터까지, 모든 종류의 컴퓨터 시스템에서 리눅스를 이용하고 있습니다. Step5 에서는 리눅스 명령어를 입력해서 컴퓨터를 제어하는 방법을 배웁니다.

- 가상 머신과 리눅스 개발환경

- 기본 명령어. 관리자 명령어

- 파일 속성

- vi, 쉘스크립트

- Telnet, OpenSSH, VNC

- Tomcat, Oracle, MySQL, MariaDB, PHP

Step 6 하둡 & 스파크

Hadoop과 Spark는 빅데이터 플랫폼을 구축하는 데 사용하는 오픈소스 빅데이터 프레임워크로, 여러 개의 컴퓨터를 하나로 묶어 대용량 데이터를 분산 처리하는 역할을 합니다.

- 하둡 개발환경 : 완전 분산 모드

- HDFS, YARN-MapReduce 분석 활용

- 하둡 Streaming, 에코 시스템

- 스파크 개발환경

- 스칼라 기본 문법, 컬렉션, 어플리케이션

- 스파크 API, 스파크 Streaming

Step 7 파이썬

파이썬은 개발자들 사이에서 널리 사용되는 매우 유연하고 강력한 언어로 특히, 빅데이터 분석 분야에서 빛을 발하는 언어입니다.

- 파이썬 개발환경

- 기본 문법, 기본 모듈

- 스크랩핑과 크롤링, Beautifulsoup

- 고급 스크랩핑, Selenium

Step 8 Django 웹 프로그래밍

Django는 Python의 무료 오픈소스 웹 애플리케이션 프레임워크입니다. Spring과 마찬가지로 Django를 이용하면 손쉽고 빠르게 웹 애플리케이션을 개발할 수 있습니다.

- 파이썬 웹 표준 라이브러리

- Django 프레임워크

- Django 템플릿 시스템

- Django 웹 서버 연동

- 외부 라이브러리 연동

Step 9 빅데이터 시각화

Step 9 에서는 Python 라이브러리인 Matplotlib, Pandas 등을 활용해 빅데이터를 시각화하는 방법에 대해 배웁니다.

- Matplotlib 기초

- 주식 실시간 데이터 시각화하기
