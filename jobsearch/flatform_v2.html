21세기 가장 유망한 직업이라는 데이터 사이언티스트.
데이터 사이언티스트는 다음 중 어떤 일을 하는 사람일까요?

1. 데이터 분석을 통해 인사이트 도출
2. 데이터 수집환경 설계 및 구축
3. 데이터 분석 플랫폼 설계 및 구축
4. 데이터 전처리 및 시각화

답은 1~4 모두 정답입니다.

데이터 사이언티스트는 데이터 분석뿐만 아니라 데이터 분석 플랫폼 구축, 입수한 데이터 품질 관리, 데이터 전처리 등
엔지니어 업무까지 모두 할 수 있어야 합니다. 이 모든 과정이 하나의 워크플로우를 이루고 있기 때문입니다.

특히 최근에는 데이터 분석가에 대한 수요가 줄고 ‘분석이 가능한 빅데이터 엔지니어’에 대한 수요가 증가하고 있습니다.
본 교육과정을 통해 데이터 사이언스 분야로 진출하세요.

* 빅데이터 플랫폼이란? * 빅데이터 플랫폼이란?

본 교육은 빅데이터 플랫폼 전문가를 양성하는 과정입니다.

수강생은 Hadoop, Spark 등을 이용해 빅데이터 플랫폼을 설계 및 구축하고,
빅데이터를 분산처리하는 법에 대해 배우게 됩니다.
또한 팀별 프로젝트를 진행하며 빅데이터 분석과 관련된 모든 과정을 실제로 경험합니다.

빅데이터 플랫폼이란?

일반적으로 데이터가 db에 저장되어 있으면 SQL 쿼리를 작성한 후 엑셀로 추출하면 됩니다.
하지만 데이터의 양이 상상하기 어려울 정도로 큰 경우 위와 같은 방식으로 데이터를 추출하면 시간이 너무나 오래 걸립니다.

대용량 데이터를 추출하기 위해 필요한 인프라가 바로 빅데이터 플랫폼입니다.
Hadoop과 Spark는 빅데이터 플랫폼을 구축하는 데 사용하는 오픈소스 빅데이터 프레임워크로,
여러 개의 컴퓨터를 하나로 묶어 대용량 데이터를 분산 처리하는 역할을 합니다.

저도 빅데이터 사이언스 분야로 진출할 수 있을까요?

네, 여러분도 할 수 있습니다.

여러 권의 프로그래밍 도서를 집필한 전문 강사님이
여러분께 프로그래밍의 기초와 빅데이터 플랫폼 설계 및 구축에 대해 체계적으로 알려드립니다.

아래와 같은 생각을 하는 분들에게 이 과정을 추천합니다.

· 데이터 업계에서 일하고 싶어서 혼자 공부했는데 한계를 느꼈어요.
Hadoop과 Spark는 독학하기 쉽지 않습니다.
그런데 국내에는 전문적으로 가르치는 곳이 거의 없죠.
본 수업을 통해 빅데이터 플랫폼에 대해 체계적으로 배워보세요.

· 데이터 분석을 공부했는데 재미있었습니다. 데이터와 관련된 모든 과정을 깊이 있게 공부하고 싶어요.
최근 빅데이터가 각광받으면서 데이터 분석을 공부하는 분이 많습니다.
하지만 분석에 사용하는 정제된 빅데이터를 어떻게 수집하고 저장하고 처리하는지 아는 분은 많지 않습니다.
본 수업에서 데이터 엔지니어링을 배우고 빅데이터 사이언스 분야로 진출하세요.

· 초보자이지만 빅데이터 사이언스 직군으로 취업하고 싶습니다. 정말 최선을 다하겠습니다.
본 과정은 프로그래밍 기초부터 빅데이터 플랫폼까지 가르칩니다.
이 모든 것을 7개월 안에 배운다는 건 벅찬 일이지만, 불가능하지는 않습니다!
확고한 의지가 있고 본인이 정말 최선을 다할 수 있다면 수업을 신청해주세요.
최선을 다해 도와드리겠습니다.
