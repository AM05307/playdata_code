사전 온/오프라인 학습시스템

비전공자 분들도 수업을 잘 따라가실 수 있도록 사전교육을 진행합니다. 사전교육은 희망자를 대상으로 무료로 진행되며 코딩을 한번도 해보지 않은 분들은 사전교육을 수강하시길 추천드립니다. 본 교육 전에 미리 프로그래밍을 배울 수 있습니다.

- [온라인 사전교육 (20일)]

- HTML5, CSS3, JavaScript

- MySQL, 알고리즘


- [오프라인 사전교육 (10일)]

- Python : 기본 문법, 데이터 타입, 연산자, 함수

Step 1 - 자바

프로그래밍의 세계에 오신 것을 환영합니다! Step1에서는 Java를 배웁니다. Java는 현재 전 세계에서 가장 많이 사용되는, 가장 중요한 언어입니다. Java로 프로그래밍 기초실력을 탄탄하게 다지세요.

- 컴퓨터의 역사

- 객체 지향 프로그래밍의 이해

- 개발 환경 구축

- 기초 문법 학습

- 미니 프로젝트

Step 2 - 데이터베이스

SQL은 데이터를 저장하고 처리하고 관리할 수 있는 핵심 언어입니다.

- 데이터베이스란?

- SQL 기본

- SQL 설치, 운영 실습

- SQL 고급

- 데이터베이스 모델링

Step 3 프론트엔드

Step3 에서는 웹 사이트의 뼈대, 살, 근육이라 할 수 있는 HTML, CSS, Javascript를 배웁니다.

- HTML로 만드는 웹 페이지 구조

- 사용자 경험 증대를 위한 CSS 활용

- 반응형 사이트를 위한 미디어 쿼리 활용

- 클라이언트 기능 구현을 위한 Javascript

- 웹 사이트 최적화

Step 4 JSP

Step4 에서는 본격적으로 웹 어플리케이션을 만듭니다. 우선 JSP/Servlet을 활용해 간단한 웹 사이트를 만듭니다.

- 웹 어플리케이션이란?

- JSP 웹 프로그래밍 기초

- Servlet 기초

Step 5 스프링

Step5 에서는 프로젝트 단위의 웹 어플리케이션 개발을 위해 스프링 프레임워크에 대해 배웁니다.

- 스프링 환경설정

- DI, AOP, DAO

- MVC

- 활용 : Sprint 기반 게시판

- MyBatis

중간 프로젝트

지금까지 배운 모든 내용을 활용해서 중간 프로젝트를 진행합니다. 스프링 부트 기반의 웹 프로젝트를 만들어본 경험은 취업할 때 큰 도움이 되실 겁니다.

- UML, ERD

- 윈도우 : 스프링 프레임워크 기반의 웹프로젝트

Step 6 AWS 클라우드

개발한 웹 어플리케이션을 일반에 공개하려면 데이터 센터(서버, 네트워크, 미들웨어, 보안 등)가 필요합니다. 이때 사용하는 것이 클라우드입니다. 클라우드를 활용해서 웹 어플리케이션을 배포할 수 있습니다.

- 로컬 데이터베이스 구축

- 클라우드 이해

- EC2를 사용한 웹 서버 구축

- 도커 컨테이너 운영 환경 구축

- 클라우드 활용
