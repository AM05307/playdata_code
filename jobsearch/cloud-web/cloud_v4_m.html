Step 1 - Java 자바

프로그래밍의 세계에 오신 것을 환영합니다!
Step 01에서는 Java를 배웁니다. Java는 전 세계에서 가장 많이 사용되는, 가장 중요한 언어입니다.
Java로 프로그래밍 기초실력을 탄탄하게 다지세요.

- 컴퓨터의 역사

- 객체 지향 프로그래밍의 이해

- 개발 환경 구축

- 기초 문법 학습

- 미니 프로젝트

Step 2 - 알고리즘과 자료구조

알고리즘은 컴퓨터가 문제를 해결할 수 있도록 단계를 설계하는 것을 의미합니다.
알고리즘은 프로그래밍의 기본이며 이미 많은 기업들이 알고리즘 코딩테스트를 통해 개발자를 채용하고 있습니다.

- 검색, 정렬 알고리즘

- 알고리즘 분석, 리스트

- 자료구조 이해

- 연결 리스트

- 스택, 큐, 정렬, 탐색

Step 3 - 데이터베이스

기업들이 웹사이트를 만드는 이유는 고객과 소통하기 위해서고 소통은 ‘데이터’로 흔적을 남깁니다.
그렇게 쌓이는 데이터의 양은 상상하기 어려울 정도로 크기 때문에 데이터베이스를 이용해 저장해야 합니다.
SQL은 데이터를 저장하고 처리하고 관리할 수 있는 핵심 언어입니다.

- 데이터베이스란?

- SQL 기본

- SQL 설치, 운영 실습

- SQL 고급

- 데이터베이스 모델링

Step 4 - 프론트엔드

Step 04에서는 웹 사이트의 뼈대, 살, 근육이라 할 수 있는 HTML, CSS, Javascript를 배웁니다.

- HTML로 만드는 웹 페이지 구조

- 사용자경험 증대 : CSS 활용

- 반응형 사이트 : 미디어쿼리 활용

- 클라이언트 기능 구현 : Javascript

- 웹 사이트 최적화

Step 5 - JSP

Step 05에서는 본격적으로 웹 어플리케이션을 만듭니다. 우선 JSP/Servlet을 활용해 간단한 웹 사이트를 만듭니다.

- 웹 어플리케이션이란?

- JSP 웹 프로그래밍 기초

- Servlet 기초

Step 6 - 스프링

Step 06에서는 프로젝트 단위의 웹 어플리케이션 개발을 위해 스프링 프레임워크에 대해 배웁니다.

- 스프링 환경설정

- DI, AOP, DAO

- MVC

- 활용 : Spring 기반 게시판

- MyBatis

중간 프로젝트

지금까지 배운 내용을 바탕으로 팀 프로젝트를 진행합니다. 프로젝트는 개발자 포트폴리오로 활용할 수 있습니다

- UML, MindMap, ER

- 윈도우 : 스프링 프레임워크 기반의 웹프로젝트

Step 7 - AWS 클라우드

개발한 웹 어플리케이션을 일반에 공개하려면 데이터 센터(서버, 네트워크, 미들웨어, 보안 등)가 필요합니다.
이때 등장하는 것이 클라우드입니다. 클라우드를 활용해서 웹 어플리케이션을 배포할 수 있습니다.

- 로컬 데이터베이스 구축

- 클라우드 이해

- EC2를 사용한 웹 서버 구축

- 도커 컨테이너 운영 환경 구축

- 클라우드 활용
