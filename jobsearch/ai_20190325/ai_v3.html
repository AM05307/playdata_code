수강생은 첫 3개월 동안 JAVA 프로그래밍과 웹 개발에 대해 배우고, 후반부 4개월 동안 Hadoop, Spark, Python을
활용한 머신러닝과 딥러닝을 공부합니다. 또한 취업 포트폴리오를 위해 딥러닝 프로젝트도 진행합니다.

머신러닝? 딥러닝?

머신러닝이란 문제 해결을 위한 코드를 작성하는 대신, 빅데이터를 알고리즘에 공급하는 방식으로 기계를 학습시켜
문제를 해결하는 기술입니다. 딥러닝은 머신러닝의 한 종류로 인공신경망을 이용합니다.
본 교육은 스파크, 텐서플로우 등을 활용해 머신러닝을 가르치며, CNN/RNN 등 딥러닝 기술도 함께 배웁니다.

여러 사람이 함께 소프트웨어를 개발하기 위해서는 소스코드의 형상관리가 필수입니다.

3. Bootstrap 특강

Backend 개발자도 기본적인 Frontend 스킬은 필수!
CSS Framework인 Bootstrap을 통해 좀 더 쉽고 빠르게
웹 화면을 구현하는 방법을 학습합니다.

3. 데모데이
