Step 1 자바 기초 프로그래밍
프로그래밍의 세계에 오신 것을 환영합니다. 프로그래밍이 처음이신 분들을 위해 Step1은 컴퓨터의 역사와 Java 언어에 대한 소개부터 시작합니다. Java는 현재 전 세계에서 가장 많이 사용되는, 가장 중요한 언어입니다. Java로 프로그래밍의 기초실력을 탄탄하게 다져보세요.

- 자바 개발환경

- 기본 문법

- GUI와 이벤트

Step 2 데이터베이스와 SQL
기업들이 웹 사이트를 만드는 이유는 고객과 소통을 하기 위해서고, 그 소통은 ‘데이터’라는 형식으로 흔적을 남깁니다. 그렇게 쌓이는 데이터의 양은 상상하기 어려울 정도로 크기 때문에 데이터베이스를 이용해 저장해야 합니다. SQL은 데이터를 저장하고 처리하고 관리할 수 있는 핵심 언어입니다.

- 오라클 개발환경

- SQL

- 서브쿼리

- JDBC
Step 3 프론트엔드와 백엔드(JSP)
STEP3에서는 웹 사이트의 뼈대, 살, 근육이라 할 수 있는 HTML, CSS, Javascript를 배웁니다. 웹 표준을 지키는 문서 구조화, 다양한 크기의 디바이스를 위한 반응형 프로그래밍, 동적 웹 사이트 제작을 위한 Javascript 등 클라이언트 개발 언어를 배웁니다. 그 다음 JSP/Servlet을 활용해 간단한 웹 사이트를 만듭니다.

- JDBC와 자바빈

- 파일 업로드, 썸네일 이미지

- EL, JSTL

Step 6 하둡 & 스파크
Hadoop과 Spark는 빅데이터 플랫폼을 구축하는 데 사용하는 오픈소스 빅데이터 프레임워크로, 여러 개의 컴퓨터를 하나로 묶어 대용량 데이터를 분산 처리하는 역할을 합니다. Step 6 에서는 Hadoop, Spark 등을 이용해 빅데이터 플랫폼을 설계 및 구축하고, 빅데이터를 분산처리하는 법에 대해 배우게 됩니다.

Step 8 머신러닝
머신러닝은 인공지능 연구의 한 분야입니다. 사람의 뇌가 진행하는 "학습 능력"을 컴퓨터로 구현하는 것이 머신러닝입니다. 샘플 데이터를 입력해서 분석한 후, 데이터에서 규칙을 찾습니다. 찾은 규칙을 바탕으로 다른 데이터를 분석하거나 미래를 예측하는 것이 머신러닝입니다.

- 선형대수, 로그, 시그마

- 스파크 머신러닝 라이브러리 ML, Mllib

- 스파크를 활용한 로지스틱

- 스파크를 활용한 의사결정트리

- 파이썬 머신러닝 프레임워크, scikit-learn

- 파이썬을 활용한 SVM

- HDFS, PySpark 연동 파이썬 머신러닝 프로그램
Step 9 텐서플로우

플레이 데이터 스페셜 프로그램!

1. 비즈니스 기획 실무 특강

모든 소프트웨어 개발의
시작은 ‘기획’입니다.
서비스 기획 실무자로부터 기획에 대해 배웁니다.

2. Github 특강

개발은 주로 팀 단위로 진행합니다.
팀 프로젝트에서 코드 버전 관리는 필수입니다.
Github 소스코드 형상관리 방법을 학습합니다.

3. Bootstrap 특강

Backend 개발자도 기본 Frontend 스킬은 필수!
CSS Framework인 Bootstrap으로
웹 화면을 구현하는 방법을 학습합니다.

4. Database 컨설팅 특강

데이터 전문기업 엔코아의 컨설턴트가
데이터 모델링에 관한
실무 특강을 직접 진행합니다.

1. 인재 추천

우수 교육생에게
엔코아 협력사 및 연계 기업
채용 기회를 우선적으로 드립니다.

2. 기업 채용 설명

기업 인사담당자에게
기업의 비전, 인재상, 분위기 등의
정보를 얻을 수 있습니다.

3. 데모데이

현직 개발자와 인사담당자가
데모데이의 심사위원으로 참석해
여러분의 프로젝트를 평가하고 시상합니다.

4. 이력서 컨설팅

현업 인사담당자가
자소서 작성법, 면접 합격 방법
등에 관해 특강을 진행합니다.
