본 교육은 빅데이터 기반 인공지능
전문가를 양성하는 과정입니다.
첫 3개월 동안에는 프로그래밍의 기초인
Java와 Spring에 대해 배우고,
이후 Python, Hadoop&Spark를
활용해 머신러닝과 딥러닝을 공부합니다.
마지막으로 취업 포트폴리오 제작을 위한
딥러닝 활용 프로젝트를 진행합니다.

머신러닝? 딥러닝?

머신러닝이란 빅데이터를 알고리즘에
공급하는 방식으로 기계를 학습시켜
문제를 해결하는 기술입니다.
딥러닝은 머신러닝의 한 종류로
인공신경망을 이용합니다.
본 교육은 Spark, Tensorflow 등을 활용해
머신러닝을 가르치며,
CNN/RNN 등 딥러닝 기술도 함께 배웁니다.

4차 산업혁명 시대에 각광받는 기술을 배워,
나만의 경쟁력을 쌓아 보세요.

이 수업을 들으면

비전공자인데
저도 할 수 있을까요?

비전공자가 본 과정을 수료하는 것은
쉬운 일이 아닙니다. 경험적으로 말씀드리면,
매일 저녁 10시까지 센터에 남아 독하게
공부하지 않으면 수업을 따라가기 어렵습니다.
만약 빅데이터 분야가 아니라, 개발자로
IT 업계에 취업하는 것이 목표라면
자바 시큐어코딩 웹 개발자 양성과정 또는
클라우드 기술 기반 웹 개발자 양성과정을
추천드립니다. 두 과정은 기초를 좀 더 깊이
가르치고, IT기업에 취업하기 위해
꼭 필요한 알고리즘을 가르치기 때문입니다.

개발자로 취업하고 싶습니다.

사전 온/오프라인 학습시스템

비전공자도 수업을 잘 따라갈 수 있도록 사전 교육을 진행합니다. 사전 교육은 희망자를 대상으로 무료로 진행되며, 코딩을 한번도 해보지 않은 분들은 수강을 권장 드립니다. 사전 교육을 통해 기초 프로그래밍을 배워보세요!

- [온라인 사전교육 / 20일]

- HTML5, CSS3, JavaScript

- MySQL, 알고리즘


- [오프라인 사전교육 / 10일]

- Python : 기본 문법, 데이터 타입, 연산자, 함수

Step 1 - 자바 기초 프로그래밍

Java는 현재 전 세계에서 가장 많이 사용되는 가장 중요한 언어입니다. Java로 프로그래밍의 기초실력을 탄탄하게 다져보세요.

- 자바 개발환경

- 기본 문법

- 클래스와 OOP

- IO

- 네트워크

Step 2 - 데이터베이스와 SQL

SQL은 데이터를 저장하고 처리하고 관리할 수 있는 핵심 언어입니다.

- 오라클 개발환경

- SQL

- 주요 함수

- 제약조건

- JDBC

Step 3 - 프론트엔드와 백엔드(JSP)

Step3 에서는 HTML, CSS, Javascript를 배우고 JSP/Servlet을 활용해서 간단한 웹 사이트를 만듭니다

- 웹 프로그램 개발환경

- 페이지 구성요소, 내장객체, 액션태그

- JDBC와 자바빈

- MVC

- 회원관리, 게시판

Step 4 - Spring

Step4 에서는 프로젝트 단위의 웹 어플리케이션 개발을 위해 스프링 프레임워크에 대해 배웁니다.

- 스프링 환경설정

- DI

- AOP

- 활용 : Spring 기반 게시판

- MyBatis

중간 프로젝트

지금까지 배운 내용을 바탕으로 팀 프로젝트를 진행합니다. 프로젝트는 개발자 포트폴리오로 활용할 수 있습니다

- UML, MindMap, ER

- 윈도우 : 스프링 프레임워크 기반의 웹프로젝트

Step 5 - 리눅스

Step5 에서는 리눅스 명령어를 입력해서 컴퓨터를 제어하는 방법을 배웁니다.

- 가상 머신과 리눅스 개발환경

- 기본 명령어

- 파일 속성

- vi, 쉡 스크립트

- Telent, OpenSSH, VNC

Step 6 - 하둡 & 스파크

Hadoop과 Spark는 빅데이터 플랫폼을 구축하는 데 사용하는 오픈소스 빅데이터 프레임워크로, 여러 개의 컴퓨터를 하나로 묶어 대용량 데이터를 분산 처리하는 역할을 합니다.

- 빅데이터

- Hadoop 개발환경 : 완전 분산 모드

- HDFS, YARN

- Spark 개발환경

- Scala 기본 문법, 컬렉션

Step 7 - 파이썬

파이썬은 개발자들 사이에서 널리 사용되는 매우 유연하고 강력한 언어로 특히, 빅데이터 분석 분야에서 빛을 발하는 언어입니다.

- 파이썬 개발환경

- 기본 문법과 모듈

- 스크랩핑과 크롤링

- beautifulsoup

- 고급 스크랩핑, Selenium

Step 8 - 머신러닝

머신러닝은 인공지능 연구의 한 분야입니다. 사람의 뇌가 진행하는 "학습 능력"을 컴퓨터로 구현하는 것이 머신러닝입니다

- 선형대수, 로그, 시그마

- 스파크 머신러닝 라이브러리 ML, Mllib

- 스파크를 활용한 로지스틱

- 파이썬 머신러닝 프레임워크, scikit-learn

- 파이썬을 활용한 SVM

- HDFS, PySpark 연동 파이썬 머신러닝 프로그램

Step 9 - 텐서플로우

텐서플로우는 머신 러닝을 개발할 떄 사용할 수 있는 오픈소스 라이브러리 입니다. 데이터를 활용해서 머신 러닝 알고리즘을 만들고, 모델을 훈련시킬 때 텐서플로우 라이브러리를 유용하게 활용할 수 있습니다.

- 텐서플로우 개발 환경

- 기본 문법 주요함수

- 선형회귀, 로지스틱회귀

- K-Means, k-Nearest Neighbors

Step 10 - 딥러닝 & 인공지능

딥러닝은 머신러닌 알고리즘의 집합입니다. 딥러닝 컴퓨터 프로그램은 빅데이터의 특성을 학습하고, 분류하고, 판별할 수 있습니다.

- 인공신경망

- 인공지능과 머신러닝, 딥러닝

- 퍼셉트론과 신경망

- 다중 퍼셉트론

- 딥러닝 활용 : 텍스트 분석

파이널 프로젝트
